# Grocery Store Management System

## Project Overview

The Grocery Store Management System is a relational database project developed using MySQL to efficiently manage grocery store operations. The database stores and manages information related to customers, employees, suppliers, products, categories, orders, payments, and order details.

The project demonstrates database design, SQL query writing, data analysis, and business insight generation using real-world retail data.

---

## Objectives

* Design a normalized relational database.
* Maintain relationships using primary and foreign keys.
* Perform CRUD operations.
* Analyze sales and customer behavior using SQL.
* Generate meaningful business insights.
